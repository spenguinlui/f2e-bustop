# 作品說明

The F2E 全台公車動態時刻查詢應用服務

# 系統說明

系統運行方式:
```
yarn install
yarn run serve
```

# 資料夾說明

* assets/images - 圖片與 icon
* assets/scss - scss 引用檔
* components - vue 元件
* json - 靜態 json 檔引用
* modules - 引用 js function 或物件
* pages - 畫面主要底版
* store - vuex 存放


# 使用技術

* Vue Cli
* Vuex
* Axios
* jsSHA
* Leaflet

# 使用第三方服務

* fontawesome
